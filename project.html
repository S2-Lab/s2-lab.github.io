Sanitizer and Mitigation

To detect various software vulnerabilities such as memory and type safety violations or logic errors accurately and efficiently, we develop various sanitizer and mitigation techniques.

Fuzzing

Fuzzing is widely used across various domains. We aim to improve this fuzzing technique further by proposing advanced fuzzing techniques that can intensively test codes prone to vulnerabilities.

Attack Surface Reduction

We develop various attack surface-reducing techniques to minimize attack surface and damages even if all vulnerabilities are not detected.

RUST Security

Rust is a new but extremely promising system programming language. We develop new techniques to detect and mitigate software vulnerabilities specific to RUST effectively.

Mobility Security

We aim to effectively detect logic errors and common software vulnerabilities (e.g., memory safety violations) that can occur in various mobility environments such as autonomous vehicles, ships, and drones.

Wed Security

We develop various Web security technics mainly to detect and eliminate various user data privacy issues (e.g., trackers) that can arise while using web-based services.

Mobile Security

We develop new techniques to effectively detect various types of malware on mobile platforms, including those generated using emerging frameworks or cross-languages.
